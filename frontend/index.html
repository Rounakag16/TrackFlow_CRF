Core Features

Lead Management

Add, stage, filter and follow-up on every prospect via Kanban or list views.

Order Workflow

Automatically convert “Won” leads into orders, track build & dispatch with status updates.

Follow-Up Reminders

Date-based alerts keep you on top of every sales and operational action item.

Dashboard & Calendar

Visualize metrics and upcoming tasks with charts and FullCalendar integration.

Your Workflow in 3 Steps

  1. Capture Leads: Fill out basic info, stage them New → Proposal Sent.
  2. Qualify & Convert: Set reminders, move to “Won” and auto-generate an Order.
  3. Fulfill Orders: Update Order status—In Development → Dispatched → Complete.

Built With

  • Backend: Node.js, Express.js
  • Database: PostgreSQL
  • Frontend: HTML, CSS, JavaScript
  • Calendar: FullCalendar.js
  • Optional: FastAPI / Django, React, Chart.js
